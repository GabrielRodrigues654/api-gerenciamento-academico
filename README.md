# API de Gerenciamento Acadêmico

## Descrição

API REST desenvolvida com FastAPI para gerenciamento acadêmico.

O sistema permite o cadastro e consulta de:

* Alunos
* Professores
* Disciplinas
* Turmas
* Matrículas
* Notas

## Tecnologias Utilizadas

* Python
* FastAPI
* SQLAlchemy
* Pydantic
* Docker

## Endpoints Disponíveis

### Home

GET /

### Alunos

GET /alunos

### Professores

GET /professores

### Disciplinas

GET /disciplinas

### Turmas

GET /turmas

### Matrículas

GET /matriculas

### Notas

GET /notas
